# Quickstart: Criando seu primeiro Lead no Connect

> **Status:** Rascunho para Validação  
> **Versão:** 1.0  
> **Template Base:** The Good Docs Project

## 1. Introdução

O **Connect** é um CRM projetado para centralizar a gestão de contatos e oportunidades de vendas. Este guia é o seu "Hello World": ele ajudará você a realizar a tarefa fundamental do sistema que é registrar um novo interesse comercial (**Lead**).

* **Público-alvo:** Novos usuários da equipe comercial e SDRs (Representante de Desenvolvimento de Vendas).
* **O que você vai aprender:** Como cadastrar um Lead manualmente e a importância da Origem.
* **Tempo estimado:** 5 minutos.

---

## 2. Antes de começar 

Certifique-se de que você possui os seguintes itens:

- [ ] Login e senha ativos para o ambiente: `https://app-dev.websupplyconnect.com/`.
- [ ] Dados de um contato real ou fictício para o teste (Nome e E-mail).

---

## 3. Passo a Passo

### Passo 3.1: Acesse a tela de Leads
Para iniciar, precisamos navegar até o módulo correto na barra lateral.
1. No menu lateral esquerdo, localize a seção **CRM**.
2. Clique na opção **Leads**.
   - *Nota: O sistema abrirá a listagem de contatos. Se for seu primeiro acesso, a lista pode estar vazia.*

### Passo 3.2: Inicie o cadastro
Agora, vamos abrir o formulário de inserção de dados.
1. No canto superior direito da tela, clique no botão **+ Novo Lead**.
   - *Isso abrirá o formulário lateral com os campos em branco.*

### Passo 3.3: Preencha as informações obrigatórias
Para que um Lead seja útil para o time de vendas, precisamos de dados mínimos.
1. No campo **Nome**, digite o nome completo do contato.
2. No campo **E-mail**, insira um endereço válido.
   > **Dica de UX:** O sistema valida o formato do e-mail em tempo real. Se o campo ficar vermelho, verifique se há espaços extras ou se falta o ponto final (Ex: .com).
3. No campo **Origem**, selecione uma das opções disponíveis (ex: "Site", "Indicação").
   - *A Origem é essencial para que o Dashboard calcule quais canais trazem mais retorno.*

### Passo 3.4: Salve e valide o registro
O passo final é garantir que o dado foi salvo corretamente.
1. Clique no botão **Salvar** ao final do formulário.
2. Verifique se apareceu uma mensagem de confirmação no topo da tela: `Lead cadastrado com sucesso!`.
3. Confirme se o novo nome agora aparece no topo da lista na tela principal de Leads.

## 5. Dúvidas Frequentes (FAQ)

**Por que não consigo salvar o Lead?** Verifique se todos os campos marcados com um asterisco (*) estão preenchidos. Geralmente, Nome, E-mail e Origem são obrigatórios.
